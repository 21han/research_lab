# research_lab

## CI/CD

[AWS CodePipeline](https://aws.amazon.com/codepipeline/) deploys any new code changes to [Elastic Beanstalk](https://aws.amazon.com/elasticbeanstalk/) automatically. We use [Slack](https://slack.com/) for deployment and for receiving notifications. 

## Testing

[Pytest](https://docs.pytest.org/en/stable/)

## Style

[Pylint](https://www.pylint.org/)

## Coverage

[Coverage](https://coverage.readthedocs.io/en/coverage-5.3/)

## Visualization

[Dash](https://plotly.com/dash/)
